## Deskripsi Project

Project ini merupakan implementasi sistem manajemen produk berbasis web yang terdiri dari 3 halaman utama:

### 1. üîê Halaman Login (`index.html`)
Halaman autentikasi untuk masuk ke sistem dengan fitur:
- Form input email dan password (NIM)
- Validasi input tidak boleh kosong
- Validasi format email
- Redirect otomatis ke dashboard setelah login berhasil
- Tampilan modern dengan gradient background

### 2. üìä Halaman Dashboard (`dashboard.html`)
Halaman ringkasan data dengan fitur:
- Sidebar menu navigasi
- 3 Card summary dengan animasi counter:
  - **Total Produk**: Menampilkan jumlah total produk (120)
  - **Total Penjualan**: Menampilkan jumlah penjualan (85)
  - **Total Revenue**: Menampilkan total pendapatan (Rp 12.500.000)
- Tombol navigasi ke halaman data produk
- Design responsif dan user-friendly

### 3. üìã Halaman List Data Produk (`products.html`)
Halaman manajemen produk dengan fitur:
- Tabel data produk dengan kolom: No, Product Name, Price, Stock, Aksi
- Data produk default:
  - Kopi Gayo - Rp 25.000 (Stock: 50)
  - Teh Hitam - Rp 18.000 (Stock: 30)
  - Coklat Aceh - Rp 30.000 (Stock: 20)
- Fitur **Edit**: Menampilkan alert dengan nama produk yang dipilih
- Fitur **Delete**: Menghapus produk dengan konfirmasi terlebih dahulu
- Format harga dalam Rupiah

## Teknologi yang Digunakan
- **HTML5**: Struktur halaman web
- **CSS3**: Styling dan layout responsif dengan Flexbox & Grid
- **JavaScript (Vanilla)**: DOM manipulation, Event Listener, dan Array methods
